# Code of Conduct

This project should be a useful, low-drama place to build a good FOSS Jellyfin music player.

Expected behavior:

- Be respectful and direct.
- Keep criticism technical and actionable.
- Assume contributors are here to improve the project.
- Do not harass, threaten, insult, or target people based on identity, background, or experience level.
- Do not post private credentials, server details, or personal data.

Maintainers may close issues, remove comments, or block participation when behavior gets in the way of the project.
