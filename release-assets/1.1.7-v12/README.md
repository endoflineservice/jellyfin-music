# Jellyfin Music 1.1.7 v12

Release APK:

- `JellyfinMusic-1.1.7-v12-release.apk`

Highlights:

- Fix Android Auto artwork delivery by serving cached album art through a local content provider.
- Prefer local artwork URIs for Android Auto browse rows, queue items, now playing metadata, and the media session.
- Add Google Play Billing support for a one-time `support_3_usd` support purchase.
- Move the support purchase card to the top of the main Settings page.
- Keep the app version name at `1.1.7` and bump Android version code to `12`.
