# Jellyfin Music 0.1.5 (version code 9)

Shareable release APK:

- `JellyfinMusic-0.1.5-v9-release.apk`

Highlights:

- Fixed a launch-time performance crash/ANR caused by smart mix generation on large libraries.
- Capped Daily/Mood mix candidate pools so Home can open quickly.
- Optimized Smart Shuffle to sample candidates instead of repeatedly sorting the full library.
- Keeps the 0.1.4 smart DJ, after-queue recommendations, and Made For You/Mood Mixes features.
