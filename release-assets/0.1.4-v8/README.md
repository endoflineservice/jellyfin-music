# Jellyfin Music 0.1.4 (version code 8)

Shareable release APK:

- `JellyfinMusic-0.1.4-v8-release.apk`

Highlights:

- Added Queue settings for AI DJ, after-queue recommendations, and fewer repeats.
- Added recommended queue handoff when the current queue finishes.
- Improved smart shuffle to use recent plays and queue history.
- Split smart cards into Made For You and Mood Mixes rows.
